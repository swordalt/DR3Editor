# DR3Editor

DR3Editor is a browser-based chart editor and creator for the rhythm game DanceRail3. It fully supports the official format, meaning everything you create is compatible with the official game. It is meant to be a replacement/alternative to the official chart editor, DRMaker.

The majority of controls and keybinds are identical to DRMaker. Click the ? button for a full list.

## Features
- Full support for the official format. (New 25-27 note types.)
- Preview Mode; top-down recreation of DanceRail3 with camera movement, NSC, Appear Mode.
- Export for DanceRail3Viewer/DR3FV.
- Audio converter (to .OGG).
- Curved note/SC generator.
- Issue finder (playability).
- Operation history (with undo/redo).
- Note organization.
- NSC helper tool.
- Note multiedit tool.
- Camera tilt manipulation tool.

## Usage
An updated build is available for use on GitHub pages [here](https://swordalt.github.io/DR3Editor/). Otherwise, you may download the project source and host it locally yourself.
